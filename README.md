# microservicegenerator

Requirements:
- access to your github project using ssh keys
- github cli installed and authorizer:
  - brew install gh
  - gh auth login
- OPENAI_API_KEY env variabele with openAI key

Engines to choose from:
OPEN_AI, OLLAMA

OPEN_AI Models to choose from:
gpt-4.5-preview
gpt-4o
gpt-3.5-turbo
